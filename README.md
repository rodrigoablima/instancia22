# Resolvedor de Bandos — D&D 2024

Ferramenta em PowerShell para acelerar combates com a regra
de Bandos do Livro do Mestre de D&D 2024.

## Recursos

- Interface gráfica (Windows Forms)
- Cálculo automático dos acertos
- Três modos de dano
- Dano médio
- Rolagem individual
- Um dano multiplicado
- Histórico
- Modelos de monstros
